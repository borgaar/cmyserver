# What?

This is a (WIP) HTTP server written in C to retrieve stats about a Unix machine.

Mainly written for myself to learn how to C.

This repository contains no C code generated by an LLM. That's right, we are talking
*handcrafted memory leaks* just like how our ancestors did it.

# Building & Running Instructions

## Prerequisites

- CMake
- GNU Make
- A C compiler

## Build -> Run
Clone the repository, then:

```bash
# Hop into the cloned repo
$ cd cmyserver

# Build the server
$ mkdir -p build && cd build && cmake .. && make

# Run it
$ ./cmyserver
```
